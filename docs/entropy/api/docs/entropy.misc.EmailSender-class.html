Class EmailSender

source code

This class implements a very simple e-mail (through SMTP) sender. It is used by the User Generated Content interface and something more.

You can swap the sender function at runtime, by redefining EmailSender.default_sender. By default, default_sender is set to EmailSender.smtp_send.

Sample code:

>>> sender = EmailSender()
>>> sender.send_text_email("me@test.com", ["him@test.com"], "hello!",
    "this is the content")
...
>>> sender = EmailSender()
>>> sender.send_mime_email("me@test.com", ["him@test.com"], "hello!",
    "this is the content", ["/path/to/file1", "/path/to/file2"])
